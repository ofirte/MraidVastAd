# MraidVastAd

Parse your Vast file and play your ads using Mraid protocol.

## Usage
Copy the HTML file to the ironSource ad simulator and watch your ad!    
https://demos.ironsrc.com/test-tool/?adUnitLoader=mraid&mode=preview&debug=
